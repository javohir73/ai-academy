## Platform-Wide Teaching Method (applies to every lesson)

1. **Top-down** — show a working result first, then explain why. Motivation before mechanism.
2. **I do → We do → You do** (gradual release). ~30% input / 70% hands-on.
3. **Math just-in-time** — visual first, formal second, "Go Deeper" optional.
4. **Zero-friction environment** — browser notebooks, datasets + GPU preloaded.
5. **AI Socratic tutor** — escalating hints, never the answer.
6. **Mastery-based progression** — a level unlocks only when its mastery check passes.
7. **Assessment ladder** — formative quiz → lesson exercise → module project → level capstone → portfolio.
8. **North-star** — % of learners who can *build AND explain* a working model.

## The Spine (one idea, deepened every level)

`model = parameters + loss + optimization`

- **L0** builds the tools · **L1** maps the field · **L2** introduces the loop · **L3** makes it deep · **L4–L5** point it at images & language · **L6** ships a product.

## Cross-Level Threads (teach in every level, deepening each time)

- **Generalization** (overfitting/underfitting) — L2 onward
- **Honest evaluation** — every level grades model quality
- **Responsible AI** — introduced L1, deepened to L6
- **"Build AND explain"** — every project requires explaining *why*
